SE CHATROOM

“Pushing Limits to Make a Simpler Version of Vevox”

Role: Tech Lead, Creating Vevox which doesn't require a password to work while managing the team and the work load.

WORKFLOW, COMMUNICATION & BRIEF

I teamed up with the rest of my team to get to know each other and then assign each person a role, schedule weekly meetings, and address the problem domain. followed by putting together the specs of the software while sketching it at the same time.

INFORMATION ARCHITECTURE & BRIEF

Before commencing the work, as a team, we had developed a rough draught of the information architecture and content for the software. One of the points in the problem domain that was discussed is having the light and dark theme modes on the software, as this will allow both users with good and bad eye sight to use the software.

FEATURES

Light/Dark Mode

SE Chatroom has a unique feature that allows users to switch between light and dark mode depending on their needs and what suits them while making use of the software.

Saves all the chats

Saving all the conversations with academics and students is what makes this software stand out.

List of channels

The software lists all the units the user is enrolled in, which makes it easier than ever before to have all the conversations with various academics and students about different units in the same software.
